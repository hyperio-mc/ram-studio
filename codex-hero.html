Inspired by Silencio.es's reference taxonomy system (godly.website) and Linear.app's dark SaaS precision (darkmodedesign.com). 5 mobile screens: category index with REF codes, doc browse grid, document viewer with code blocks, taxonomic search, and team contributor map. PRODUCT BRIEF

OVERVIEW

CODEX is a private team knowledge base for engineering teams — think Linear meets Notion but designed with the archival precision of a Swiss reference system. Every document has a REF: code. Every category is indexed. Documentation becomes a living, searchable vault that engineers actually use.

TARGET USERS

  • Engineering teams of 5–50 at early-stage and growth startups
  • Platform/infrastructure engineers managing runbooks and ADRs
  • Tech leads maintaining API references and architecture decisions
  • New engineers onboarding who need a single source of truth

CORE FEATURES

  • REF taxonomy system — Every doc gets a CDX-XXX identifier for instant citation
  • Category Index — Dashboard view of all knowledge domains with health scores and doc counts
  • Doc Browser — Filtered card grid with tag taxonomy and recent/popular sorting
  • Doc Viewer — Editorial document view with inline code blocks and table of contents
  • Smart Search — Full-text search with taxonomy filters and amber-highlighted keyword matches
  • Team Map — Contributor health scores, activity charts, and expertise heatmap

DESIGN LANGUAGE

  • Swiss archival editorial — Inspired by Silencio.es's reference taxonomy system (godly.website)
  • Dark SaaS precision — Deep near-black (#0C0C12) with cool undertones, influenced by Linear.app (darkmodedesign.com)
  • Dual accent system — Violet (#7060F0) for UI states and interaction; Amber (#E8A838) for archival REF codes and keyword matches
  • ALL CAPS label system — All metadata labels are uppercase with wide tracking; creates authoritative, archival feel
  • Monospace typography — Reinforces developer-first positioning; every element feels like it belongs in a terminal

SCREEN ARCHITECTURE

  • INDEX — Category dashboard: 6 knowledge domains each with REF codes, doc counts, update timestamps, health progress bars
  • BROWSE — Doc card grid with hot/trending indicator, tag pills, author avatars, ref codes, recency labels
  • DOC VIEW — Full document: breadcrumb, table of contents with active state, endpoint cards (GET/POST/DELETE), syntax-highlighted code block with copy button
  • SEARCH — Accent-bordered search input, taxonomy filter chips, result rows with amber keyword match highlights, action row per result
  • TEAM — Vault health score (94/100), 14-day activity bar chart, contributor rows with avatar, role tags, streak badges, expertise health bars
