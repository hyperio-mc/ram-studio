## Radiant — Production Shaders for the Web
**Source:** https://x.com/pbakaus/status/2034410464424382824 (Paul Bakaus, Mar 18 2026)
**Site:** https://radiant-shaders.com
**GitHub:** https://github.com/pbakaus/radiant
**Added:** 2026-03-18

### What it is
80+ production-ready WebGL + Canvas 2D shaders. Single self-contained HTML files. 0 dependencies. MIT license. Pick, copy source, drop in, ship.

### Notable shaders
- **Event Horizon** — physics-based black hole, raytraced gravitational lensing, volumetric accretion disk
- **Gilded Fracture** — Kintsugi-inspired golden cracks on dark surface, molten gold light bleed
- **Kinetic Grid** — spring-connected grid mesh, tension-colored connections in cyan + magenta
- **Flow Field** — particles following Perlin noise with warm amber trails
- **Torn Paper** — paper tearing apart with fibrous edges, volumetric light underneath
- **Silk Cascade** — flowing layered silk, anisotropic specular highlights

### Categories
- Style: Full canvas (64), Standalone objects (19), Particles (17), Physics (22), Noise (25), Organic (20), Geometric (29)
- Technique: WebGL (57), Canvas 2D (26)

### Color themes (per shader)
Amber, Mono, Blue, Rose, Emerald, Arctic — switchable per shader

### How to use
1. Browse gallery
2. Copy single HTML file source
3. Paste into AI coding tool to remix colors/timing
4. Drop into project — no npm, no build, no bundler

### RAM Studio applications
- **Hero page backgrounds** — use a full-canvas shader as animated bg behind the pen thumbnails
- **Landing page** — replace static dark bg with a live particle/noise shader
- **Per-design accent** — SIGNAL could use Kinetic Grid (cyan+magenta → swap to chartreuse), PURA could use Silk Cascade (warm), GRID could use Event Horizon
- **Viewer backgrounds** — subtle noise shader behind the pen viewer
