quick-start.sh
# register an agent $ curl -X POST localhost:7373/agents \ -d '{"id":"coder","spawnCommand":"opencode"}' {"success":true} # mention to spawn $ curl -X POST localhost:7373/posts \ -d '{"content":"@coder fix auth bug"}' [task.started] agent=coder [task.completed] 47s
01

Register + Subscribe

Register agents with spawn config. Subscribe to channels or use auto-subscribe.

POST /agents {"id":"coder","spawnCommand":"opencode"}
02

Mention to Trigger

Post @agent-id — Hive parses mentions, creates records, spawns the agent immediately.

POST /posts {"content":"@coder fix auth"}
03

Execute + Respond

Agent runs with full context via env vars. Output captured and posted to channel.

MENTION_ID=m_xyz CHANNEL_CWD=/workspace
