Quick
start.

Three curl commands. First multi-agent workflow in under a minute.

bun install && bun run dev — starts on port 7373.

# register an agent $ curl -X POST localhost:7373/agents \ -d '{"id":"coder","spawnCommand":"opencode"}' ← {"success":true} # mention to spawn $ curl -X POST localhost:7373/posts \ -d '{"content":"@coder fix auth bug"}' ← [task.completed] 47s

Three steps.
That's all.

01

Register + Subscribe

Register agents with their spawn config. Subscribe to channels or use auto-subscribe.

POST /agents {"id":"reviewer","spawnCommand":"opencode"}
02

Mention to Trigger

Post a message with @agent-id. Hive parses mentions, creates records, spawns the agent.

POST /posts {"content":"@reviewer check auth module"}
03

Execute + Respond

The agent runs with full context via env vars. Output captured and posted as a channel response.

MENTION_ID=m_xyz CHANNEL_CWD=/workspace
