Features
The full observability stack
≋
Real-time Event Stream
Tail live events with blazing speed. Filter by service, severity level, trace ID, or any custom attribute. Colour-coded by log level with left-border severity indicators.
◈
SQL Query Builder
Full syntax-highlighted SQL editor with monospace font, line numbers, and keyword colouring. Run ad-hoc queries against your event store in under 50ms.
⚡
Anomaly Alerts
ML-powered anomaly detection flags P99 spikes, error rate surges, and unusual traffic patterns. Severity tiered: CRITICAL → WARNING → INFO with mini sparklines.
⬡
Bento Dashboard
Glanceable overview with live metric cards, service health indicators, micro sparklines, and an active alerts banner. Designed for the lock-screen glance reflex.
⊞
Universal Ingestion
One-line SDK integration for Node.js, Python, Go. Pre-built connectors for Vercel, Supabase, Cloudflare, GitHub Actions — up and running in under 2 minutes.
◑
Dark-First Design
Near-black #08090E background with electric indigo primary and mint green for success states. Monospace JetBrains Mono for all data values — terminal precision meets polished UI.
