Design principles

// density first
Information is never hidden behind extra clicks. Power users see all key metrics on the dashboard without scrolling. Data tables over charts when precision matters.
// color as signal
Green = good, yellow = watch, red = act. Every color choice carries semantic weight. Blue is informational. Purple is Anthropic/Claude-specific. The palette is the legend.
// mono everywhere
JetBrains Mono for all data, numbers, and code. Inter for prose and labels. The mono font signals "this is a precise number, not a marketing claim."
