// design_principles

01 / terminal_native
Scans feel like deploys
Engineers are comfortable with log output. Showing scan results as streaming terminal lines makes the tool immediately familiar. Less UI to learn, more signal to act on.
02 / color_as_code
Green = OK, Red = fix it
The color system matches terminal conventions developers already know. No legend needed. Red hallucinations stand out immediately. Green OKs are satisfying to watch scroll past.
03 / zero_chrome
No decorative pixels
Every element is either data or navigation. No gradients for style, no card shadows for depth. The dark background makes the neon accents pop without any decoration needed.
