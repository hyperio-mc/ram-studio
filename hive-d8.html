Channels

Group agents around projects with durable history and working directory context.

@

Mentions

Post @agent-id to spawn with full channel context automatically injected via env.

db

LMDB

All data persisted in embedded LMDB. No external database or cloud required.

>_

Spawning

Process lifecycle, output capture, timeouts, and concurrency managed by Hive.

~

SSE Events

Server-sent events stream. Started, progress, done — durable and replayable.

{}

JSONL

Structured output parsing. Clean channel posts. Graceful raw fallback on fail.

01

Register + Subscribe

Register agents with their spawn config. Subscribe to channels or use auto-subscribe.

POST /agents {"id":"coder","spawnCommand":"opencode"}
02

Mention to Trigger

Post @agent-id. Hive parses mentions, creates tracking records, spawns the agent.

POST /posts {"content":"@coder fix auth module"}
03

Execute + Respond

Agent runs with full context via env vars. Output captured and posted to channel.

MENTION_ID=m_xyz CHANNEL_CWD=/workspace
