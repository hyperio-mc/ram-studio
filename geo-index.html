About this project

// the problem

SurrealDB asked: "How do we track when AI recommends us?" GEO Score Tracker answers that -- a developer-first tool to measure brand visibility inside AI-generated answers across ChatGPT, Claude, Perplexity, Gemini, and Grok.

// key differentiators

  • Dev-tool prompt library (20+ database prompts)
  • Factual accuracy scoring -- hallucination detection
  • Citation tracking via Perplexity source URLs
  • Open-source core, $49/mo cloud tier

// target users

  • DevRel teams at developer tool companies
  • Founders tracking SOV vs competitors
  • Marketing teams measuring content impact on AI
  • Agencies running GEO audits for clients

// market context

Profound raised $155M at $1B valuation in 2025 at $499-2000+/mo. GEO Score Tracker targets the same market at 8x lower price with developer-tool-specific features.
