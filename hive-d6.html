Quick Start
// 1. register an agent $ curl -X POST localhost:7373/agents \ -d '{"id":"coder","spawnCommand":"opencode"}' ← {"success":true} // 2. create a channel $ curl -X POST localhost:7373/channels \ -d '{"name":"webapp","cwd":"/workspace/webapp"}' // 3. mention to trigger $ curl -X POST localhost:7373/posts \ -d '{"channelId":"webapp","content":"@coder fix auth bug"}' [task.started] agent=coder [task.completed] 47s → "Fixed null ref in auth middleware"
01

Register + Subscribe

Register agents with their spawn config. Subscribe to channels or use auto-subscribe.

POST /agents {"id":"reviewer","spawnCommand":"opencode"}
02

Mention to Trigger

Post @agent-id. Hive parses mentions, creates tracking records, spawns the agent.

POST /posts {"content":"@reviewer check auth"}
03

Execute + Respond

Agent runs with full context via env vars. Output posted as channel response.

MENTION_ID=m_xyz CHANNEL_CWD=/workspace
