Original Design Challenge

"Design a dark AI threat intelligence & security operations platform for "STYX" inspired by: 1. Evervault.com/customers (godly.website) — deep cosmic near-black #010314 background, Evervault violet #6633EE accent, cool off-white #DFE1F4 text. Security-forward SaaS aesthetic with glow halos and precise information density. 2. Superset.sh (darkmodedesign.com) — terminal-embedded live data feeds, parallel agent orchestration UI, real-time activity panels baked into the interface. 3. Darknode.io (Awwwards) — dark AI automation agency aesthetic with kinetic data streams and cosmic glow system. 5 mobile screens: Command Center with live threat gauge + terminal feed, Real-time Threat Feed with severity filtering, Incident Deep-dive with attack timeline + AI analysis, Playbooks with running workflow progress, and Threat Intel with actor profiles + IOC database."

Product Brief

Overview

STYX is an AI-powered threat intelligence and security operations platform that detects, attributes, and automatically responds to cyber threats in real-time. Named after the mythological boundary between worlds, STYX sits at the perimeter between your infrastructure and adversaries — an always-on SOC powered by AI. The design draws directly from Evervault's deep cosmic palette (discovered on godly.website) and Superset.sh's terminal-embedded live data feed aesthetic (darkmodedesign.com).

Target Users

  • Security Operations Center (SOC) Analysts — monitoring the live threat feed and investigating incidents
  • Incident Responders — executing playbooks and containing active threats
  • CISOs — reviewing the command center's threat index and risk posture
  • Threat Intelligence Analysts — maintaining actor profiles and IOC databases

Core Features

  • Command Center — Real-time threat index gauge (0–100 risk score), 7-day trend bars, live terminal feed, 4 KPI metric cards, AI insight panel, quick-action chips
  • Threat Feed — Chronological event stream with severity filtering (Critical/High/Medium/Blocked/AI Only), attacker geolocation, auto-block confirmation
  • Incident Deep-dive — Full attack timeline with AI attribution confidence, attacker profiling, coordinated response actions, AI-generated containment brief
  • Playbooks — Automated response workflows with real-time progress visualization, step completion tracking, live run counter
  • Threat Intelligence — Curated actor database with threat scores, IOC counts, AI confidence ratings, TTPs, and attribution

Design Language

Deep cosmic near-black (#010314) — Evervault's signature background — creates an infinite-depth canvas that makes the vivid violet (#6633EE) accent pop. The glow halo system (inspired by Evervault's particle aesthetic) layers multiple ellipses at decreasing opacity to create ambient light sources. The terminal feed component (Screen 1 & 2) is a direct homage to Superset.sh's embedded code-editor hero. Severity colors form a strict hierarchy: Cyan=Live, Green=Safe/Blocked, Amber=Warning/High, Red=Critical/Active threat. Cool off-white (#DFE1F4) for body text prevents eye strain during long SOC sessions.

Screen Architecture

  1. Command Center — Risk gauge + 7-day trend chart, 4 KPI cards, live terminal feed panel (macOS-style), quick action chips, AI insight card
  2. Threat Feed — Severity filter chips, chronological threat cards with colored severity bars, attacker attribution + geolocation
  3. Incident Detail — INC reference header, 5-step attack timeline with AI attribution event, attacker profile table, response action buttons, AI analysis card
  4. Playbooks — Running/Ready status badges, step completion chips, progress bar with live indicator, run statistics
  5. Threat Intel — 4-metric stats row, actor cards with threat scores, TTPs tag chips, IOC counts, AI confidence ratings
